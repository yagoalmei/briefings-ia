01

Anthropic libera Opus 4.7 + integrações Microsoft 365 + Claude Design

Em 5 de maio, a Anthropic anunciou o Claude Opus 4.7 com 64,37% no benchmark Vals AI Finance Agent, lançou dez templates de agentes para serviços financeiros, e estreou integrações nativas com Microsoft 365 (Excel, PowerPoint e Word, com Outlook em breve). Também lançou o Claude Design, ferramenta experimental que transforma descrições em linguagem natural em designs, apresentações e protótipos.

Por que importa

Workflow de finance/ops integra Claude direto nos apps Office sem orquestração intermediária — é a primeira vez que uma AI lab tem add-ons oficiais da Microsoft side-by-side com Copilot. Claude Design abre frente nova competindo com Figma+v0+Lovable. Pra quem usa Anthropic em produto, vale revisar prompts financeiros aproveitando o boost do Opus 4.7.

Fonte: Anthropic Release Notes

02

Anthropic dobra limites de Claude Code e API com compute da SpaceX/xAI

Em 6 de maio, a Anthropic dobrou os limites de uso de 5 horas para planos Pro, Max, Team e Enterprise do Claude Code e Claude API, e removeu reduções de horas de pico. Habilitado por parceria com a SpaceX que dá acesso à totalidade do Colossus 1 (220.000 GPUs NVIDIA, 300+ MW) em até um mês.

Por que importa

Efeito prático imediato: quem usa Claude Code em produção ganha 2x mais capacidade sem upgrade de plano. Times que tinham throttling em horas de pico (especialmente fuso UTC-3 vs operações globais) recuperam previsibilidade. Sinaliza também que a Anthropic não vai ser limitada por compute no próximo ciclo — diferencial competitivo vs labs que dependem só do AWS/Azure/GCP.

Fonte: JLS Blog

03

OpenAI promove GPT-5.5 Instant como default — 52% menos hallucinations

Em 5 de maio, OpenAI substituiu GPT-5.3 Instant pelo GPT-5.5 Instant como modelo default do ChatGPT para todos os usuários. Em avaliações internas, o novo modelo produziu 52,5% menos alegações alucinadas em prompts de alto risco (medicina, direito, finanças).

Por que importa

Quem usa ChatGPT em workflows críticos ganha qualidade sem upgrade de plano. Setores high-stakes (jurídico, financeiro, médico) que evitavam usar AI default por medo de alucinação têm agora um bar mais alto. Pra produtos construídos sobre ChatGPT API: vale re-rodar evals porque o comportamento mudou — alguns prompts podem precisar ajuste.

Fontes: OpenAI, TechCrunch

04

ChatGPT lança Memory Sources com controle de transparência

OpenAI introduziu Memory Sources em todos os modelos do ChatGPT. Quando uma resposta é personalizada, o usuário vê exatamente qual contexto foi usado (memórias salvas, chats anteriores) e pode deletar ou corrigir o que está desatualizado.

Por que importa

É a primeira vez que uma big tech entrega transparência operacional sobre personalização de AI consumer. Resolve dois problemas simultâneos: privacidade (controle do usuário) e qualidade (memórias erradas degradam resposta). Para legal/compliance: facilita defesa em GDPR/LGPD ("usuário tem visibilidade"). Vai forçar Google/Anthropic a responder com features equivalentes.

Fonte: ChatGPT Release Notes

05

ChatGPT Workspace Agents disponível para Enterprise com EKM

ChatGPT Workspace Agents agora estão disponíveis em workspaces ChatGPT Enterprise com Enterprise Key Management. Permite construir e usar agentes para tarefas repetitivas e workflows de negócio através de apps conectados, incluindo ChatGPT e Slack.

Por que importa

Esse era o bloqueador final pra grandes empresas adotarem ChatGPT como camada de automação — IT/security exigia controle de chaves próprias. Com EKM, a tese de "ChatGPT como sistema operacional de workflows" fica concreta. Para quem vende SaaS B2B: integração com ChatGPT via apps conectados é canal de distribuição novo. Pra times de RevOps/SalesOps: vale prototipar agente de qualificação de lead.

Fonte: ChatGPT Enterprise Release Notes

06

ChatGPT for Clinicians estreia versão gratuita verificada com CME credits

OpenAI lançou ChatGPT for Clinicians, versão gratuita para clínicos verificados nos EUA, com clinical search, citations, reusable skills, deep medical research, documentation help e CME credits (educação médica continuada).

Por que importa

É a estratégia de OpenAI saindo do horizontal "AI pra todos" pro vertical "AI especializada com loop de credibilidade". Free + CME credits cria flywheel: médicos usam, ganham créditos pra licença, viram advogados do produto. Sinaliza que big techs vão atacar verticais regulados (health, legal, finance) com versões gratuitas + credenciamento — vai pressionar quem vende SaaS vertical "AI-powered" sem essas pontes.

Fonte: ChatGPT Release Notes
